# dos 指令

| #盘切换         | '>'+直接输盘号                                         |
| --------------- | ------------------------------------------------------ |
| #查看文件       | dir                                                    |
| #切换目录       | cd /d                                                  |
| #返回上一层     | cd..                                                   |
| #清理屏幕       | cls                                                    |
| #退出终端       | exit                                                   |
| #查看电脑的ip   | ipconfig                                               |
| **## 文件操作** |                                                        |
| #创建文件夹     | md                                                     |
| #创建文件       | 直接cd>文件名                                          |
| #删除文件       | del                                                    |
| #移除目录       | rd                                                     |
| #重命名         | ren xxx.txt yyy.txt    xxx为原文件名 yyy为更改后文件名 |
| #删除非空文件夹 | rd/S                                                   |
|                 |                                                        |
